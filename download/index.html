IN 01

Farming Simulator

Wire Vantix Radio into FS25's custom internet radio list.

  1. 📁Open the internet radio XML file
  2. 🎧Add a stream entry pointing at Vantix
  3. 🎮Select it from the in-game radio list
Path
Documents\My Games\FarmingSimulator2025\music\streamingInternetRadios.xml
Add this line
<streamingInternetRadio href="https://eu8.fastcast4u.com/proxy/vantixradio/stream" />
IN 02
🚛

ETS2 / American Truck Simulator

Add Vantix Radio to SCS Software games via live_streams.sii.

  1. 📄Open live_streams.sii
  2. 🎧Add a stream entry in the list
  3. 🚚Find it in the in-game radio list
Path
Documents\Euro Truck Simulator 2\live_streams.sii
Add this line
stream_data[0]: "https://eu8.fastcast4u.com/proxy/vantixradio/stream|Vantix Radio|Variety|EN|128"

Already have stations installed? Bump the index up — stream_data[1], [2], and so on.
